# Cinematic Login Experience

A visually stunning, interactive 3D login page built with React, Three.js, and Tailwind CSS.

## Features

- **3D Scene**: Immersive background with interactive elements.
- **Cinematic Transitions**: Smooth animations and glassmorphism effects.
- **Responsive Design**: Works beautifully on all devices.
- **Modern Tech Stack**: React, Vite, Framer Motion, and shadcn/ui.

## Getting Started

1. Clone the repository.
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the development server:
   ```bash
   npm run dev
   ```

## Design

The design focuses on a "cinematic" feel, using depth, motion, and high-quality typography to create a memorable user experience.
